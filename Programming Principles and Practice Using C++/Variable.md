# 🧠 1. Variableன்னா என்ன?

👉
**Variable = memoryல data வைச்சுக்குற பெயருள்ள இடம்**

---

## 🎯 Example

```cpp
int age = 42;
```

👉

* `age` → variable name
* `42` → value
* `int` → type

---

## 📦 Visual

```text
[ age ] → 42
```

---

# 🧩 2. Object vs Variable

👉

* Object → memory இடம் 📦
* Variable → அந்த objectக்கு பெயர் 🏷️

---

👉
**Named object = Variable**

---

# 🧠 3. Type என்ன?

👉
**Type → என்ன data store பண்ணலாம் என்பதை define பண்ணும்**

---

## 🎯 Example

```cpp
int age = 42;
std::string name = "Vasanth";
```

---

👉

* `int` → numbers மட்டும்
* `string` → text மட்டும்

---

# ⚠️ 4. Wrong type → error

---

```cpp
std::string name = 39;   // ❌ error
int steps = "Hello";     // ❌ error
```

---

👉
**compiler type check பண்ணும் 🔥**

---

# 🧠 5. Value என்ன?

👉
**variable உள்ளே இருக்கும் actual data**

---

## Example

```cpp
int x = 10;
```

👉 value = 10

---

# 🧩 6. Definition என்ன?

👉
**variable create பண்ணும் statement**

---

## Example

```cpp

```

👉 இது:

* define பண்ணுது
* memory allocate பண்ணுது

# 🎯 7. Initialization என்ன?

---

👉 `10` = initializer
