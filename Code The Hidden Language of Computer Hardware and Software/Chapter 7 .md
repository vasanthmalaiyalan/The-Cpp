# 7️⃣ Feature 1 — Positional system

Digit value depends on **position**.

Example:

```
4825
```

Meaning:

```
4 thousands
8 hundreds
2 tens
5 ones
```

Mathematically:

```
4825
= 4 × 1000
+ 8 × 100
+ 2 × 10
+ 5 × 1
```

Or:

```
4 × 10^3
+ 8 × 10^2
+ 2 × 10^1
+ 5 × 10^0
```
