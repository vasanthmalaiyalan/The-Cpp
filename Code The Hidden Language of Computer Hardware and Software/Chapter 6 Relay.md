# 1️⃣ Relay = ON / OFF switch

Relay ஒரு switch போல வேலை செய்கிறது:

```
Signal வந்தால் → switch ON
Signal இல்லையெனில் → switch OFF
```

Computer language-ல்:

```
ON  = 1
OFF = 0
```

---

# 2️⃣ இரண்டு Relay சேர்த்தால் Logic செய்யலாம்

## AND Logic (இரண்டும் ON என்றால் தான் ON)

Idea:

```
Relay A   Relay B   Output
```

Diagram idea (series connection):

```
Battery
  │
 [Relay A]
  │
 [Relay B]
  │
 Light
```

### எப்படி வேலை செய்கிறது

| A | B | Light |
| - | - | ----- |
| 0 | 0 | OFF   |
| 0 | 1 | OFF   |
| 1 | 0 | OFF   |
| 1 | 1 | ON    |

அதாவது:

```
A AND B
```

---

# 3️⃣ OR Logic (ஏதாவது ஒன்று ON என்றால் ON)

Relay parallel connection:

```
Battery
  │
 ├──[Relay A]──┐
 │             │
 └──[Relay B]──┘
        │
       Light
```

### Result

| A | B | Light |
| - | - | ----- |
| 0 | 0 | OFF   |
| 0 | 1 | ON    |
| 1 | 0 | ON    |
| 1 | 1 | ON    |

இதுதான்:

```
A OR B
```

---

# 4️⃣ NOT Logic (reverse)

ஒரு relay-யை normally closed switch போல பயன்படுத்தலாம்.

```
Input → Relay → Output
```

| Input | Output |
| ----- | ------ |
| 0     | 1      |
| 1     | 0      |

அதாவது:

```
NOT
```
